安装Apache服务器实例分享

发表于 2017-10-26 | 分类于 环境搭建 | | 阅读次数:

前言

本文主要是记录安装过程中踩到的坑，快点跟着我的教程来，让你轻轻松松部署Apache服务器~！

下载Apache安装包

http://www.apachelounge.com/download/

解压安装包

修改文件

打开C:\Apache\Apache24\conf目录下的httpd.conf文件并作以下修改：

  • ServerRoot（apache目录）改为C:\Apache\Apache24

  • ServerName （服务器名称） 去掉#，改为127.0.0.1

  • DocumentRoot （修改网站根目录地址）自己设置，一般设置为C:\Apache\Apache24\htdocs

  • 与上一步地址相同

  • ScriptAlias 改为 /cgi-bin/ “apache目录+/cgi-bin/“

  • （与上面那个Directory 不同）

安装卸载命令：每改一次.conf文件，都要卸载后重新安装

安装/卸载命令（以管理员身份运行CMD）
安装命令：httpd -k install -n apache
卸载命令：httpd -k uninstall -n apache

启动Apache服务

开始–>运行–>services.msc

开启不了服务，就进入C:\Apache\Apache24\bin，双击ApacheMonitor.exe，点击start

错误解决

端口占用

这是因为80端口已经被用了，所以你就要改端口，还是在httpd.conf文件中找到Listen 80这一行（大概在58行），将80改为8080就行了，然后再卸载，安装。
